3️⃣ What is the difference between map(), filter(), and forEach()?

Ans : The difference between map(), filter(),  and forEach () is

map() :  It applies a specific function to each item in an array and returns a new array with the results.

filter() : It checks each item in the array. If the condition you provide is true, then that item is placed in the new array; otherwise, it is not. 
 
forEach(): It runs a function once for each item in the array, but it does not return a new array itself


4️⃣ What is an arrow function?
Ans : Arrow functions are a new addition to JavaScript ES6 that essentially acts as a kind of (syntactic sugar) to make code more beautiful.​It works like a regular function expression, but its writing style or syntax is slightly different from that of regular functions.
